# USACO 2020 December Contest, Bronze: Problem 1 - Do You Know Your ABCs?

[Problem Link](https://usaco.org/index.php?page=viewproblem2&cpid=1059)

## Problem Description

Elsie possesses three positive integers A, B, and C where A ≤ B ≤ C. She provides Bessie with seven numbers that represent A, B, C, A+B, B+C, C+A, and A+B+C in some unknown order. The task is to determine the original values of A, B, and C from these seven numbers. The solution is guaranteed to be unique.

## Input Format

A single line containing seven space-separated integers.

## Output Format

Three space-separated integers representing A, B, and C.

## Sample Input

```
2 2 11 4 9 7 9
```

## Sample Output

```
2 2 7
```

## Scoring Notes

- Test cases 2-3: C ≤ 50
- Test cases 4-10: No additional constraints
